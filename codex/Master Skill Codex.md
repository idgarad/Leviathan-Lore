# Master Skill Codex


## Introduction
- Overview of the Skill System
- Explanation of Suites (Skill Groups)
- Skill Ranks (1-5), XP scaling, and diminishing returns


### XP Progression
- Base XP cost for Rank 1: **1000 XP**
- Each subsequent rank costs **2x** the previous rank (e.g., Rank 2 = 2000 XP, Rank 3 = 4000 XP, etc.)
- Skill Types and Multipliers:
   - **Baseline skills:** x1 XP cost
   - **Spec line skills:** x2 XP cost
   - **Efficiency line skills:** x3 XP cost
- All skills are subject to diminishing returns at higher ranks.

---

## Effect Abbreviations

- **(M)**: Multiplicative effect. Each bonus is applied to the already-modified value, resulting in diminishing returns.
- **(A)**: Additive effect. Each bonus is summed directly for a linear increase.

Use these abbreviations in skill descriptions and effect listings throughout the codex.

## Suite: Qualifier (Template)

### Suite Description
Qualifier Suites are special sets of skills required to be licensed to operate a ship of a specific class. Each ship class has its own Qualifier Suite, and all skills within are considered Spec line skills (x2 XP multiplier). These skills serve as gatekeeping certifications for command, operation, and specialized functions aboard that class of vessel.
